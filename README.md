This folder contains a collection of Python programs completed as part of the Code2Xplore – 60 Days Challenge conducted by the Department of Computer Science and Engineering at SRM University–AP.
The purpose of this challenge is to strengthen programming fundamentals, improve logical thinking, and gain hands-on coding experience through daily problem-solving.

Objective-:
Practice Python programming daily

Improve problem-solving skills

Understand core programming concepts

Build consistency in coding

Develop clean and structured code
